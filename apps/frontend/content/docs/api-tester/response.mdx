## Response

**Response** is what the API returns after you click **Send**. The response panel shows it in two tabs: **Response** (body) and **Headers**.

### Response panel

| Field | Description |
|-------|-------------|
| **Status** | HTTP status code and text (e.g. `200 OK`, `404 Not Found`, `500 Internal Server Error`). |
| **Duration** | Request duration in milliseconds. |
| **Response tab** | Raw response body (JSON, HTML, or plain text). For HTML you can switch to a preview. |
| **Headers tab** | All response headers (read-only), e.g. `Content-Type`, `Cache-Control`. |

If the request fails (network error, CORS, or no server), an error message is shown instead of a response body. The **Details** tab in the request panel shows expected **response types** from route meta (e.g. shape for 200 vs 400).

|-----|--------------------|

Status might be **200 OK** and duration **45 ms**. For **404 Not Found**, the body might be `{ "error": "User not found" }`.
